Tutorial, basico de Bases de Datos, Profesor MVP Juan Carlos Gilarnaz.

Las bases de datos son un conjunto de tablas, las tablas a su vez se conforman por registros y campos;
se puede hacer una analogia entre las tablas de bases de datos y las tablas de excel (hoojas de calculo), donde los REGISTROS son filas y los CAMPOS son el equivalente a las columnas.

 
Modelo relacional: permite relacionar las tablas dentro de la base de datos.

Entidades, claves primarias, claves externas y relaciones.


una entidad es una tabla.

Una entidad es la representacion de un objeto o concepto dle mundo real, por ejemplo: un almacen.

Las claves: funcionan como el indice de un libro, de las diferentes partes que componen el objeto o entidad.

Claves primarias identifican cada registro, las claves externas es un campo de una tabla relacional que coincida con la columna.


Las relaciones pueden ser uno a uno, uno a varios y varios a varios.
